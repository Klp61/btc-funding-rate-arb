# btc-funding-rate-arb

Project Summary

This project investigates whether funding rate signals in BTC/USDT perpetual futures contain exploitable carry alpha, and if so, under what market microstructure conditions that alpha is best captured. 

The research pipeline spans data acquisition, feature engineering, carry alpha analysis, adverse selection estimation, strategy optimisation, and backtesting — built end-to-end in Python using Polars and NumPy, divided into modules.

A full writeup and explanation of analysis and results can be found in WriteUp.pdf which can be opened in a PDF viewer or in a suitable browser, e.g. Google Chrome.
